3. Color Guide
A
Blue: Correct letter! Input moves to next box.
B
Yellow: This letter is in the word, but NOT here.
C
Red: This letter is not in the word at all.
E
Dashed Line: Appears when a letter exists more than once. Disappears when all are found.
